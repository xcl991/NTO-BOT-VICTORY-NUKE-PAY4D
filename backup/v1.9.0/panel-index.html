Panduan Command

Format Command Telegram:

CUTT.LY (shortlink):

cutt.ly GANTI NOMOR
old mst:62896688686
new mst:62898978987

NUKE (identity page):

Captain77 GANTI NOMOR
old mst:62896688686
new mst:62898978987

Keduanya (cs + mst):

cutt.ly GANTI NOMOR
old cs:62896688686
new cs:62811112222
old mst:62896688686
new mst:62898978987

Multi-command (pisah ---):

cutt.ly GANTI NOMOR
old mst:62896688686
new mst:62898978987
---
Captain77 GANTI NOMOR
old cs:676767
new cs:646464
  • old cs / new cs = nomor lama → baru WhatsApp CS
  • old mst / new mst = nomor lama → baru WhatsApp MST
  • Wajib ada pasangan old + new (minimal salah satu: cs atau mst)
  • Old number diverifikasi — jika tidak cocok, perintah dibatalkan
  • Multi-command --- diproses berurutan

NUKE: tulis nama akun (misal Captain77) — harus cocok dengan akun GANTINOMOR NUKE di database. Bot navigasi ke /configuration/identity, verifikasi old number, ganti, dan submit.

CUTT.LY: tulis cutt.ly sebagai nama (bukan nama akun). 1 akun CUTT.LY dipakai untuk semua panel. Bot search old number di shortlink → ganti semua link yang cocok (termasuk pagination).
